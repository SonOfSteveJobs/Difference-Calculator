## Description:
This tool finds the difference between two files in JSON or YAML format. The output can be generated in 3 different ways: stylish, plain or json.

## Requirements:

- Node.js 18 

## Setup:
Clone the project
```
$ git clone 
```
Setup with command:
```
$ sudo make setup
```
